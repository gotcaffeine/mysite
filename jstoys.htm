Some CSS/Javascript/Jquery doodads

Lately I've been having fun with creating colorful HTML without the use of art images (code only). I was playing around with different ways of using random colors on CSS elements. Here's some of my toys. (Uses Jquery and jquery Color). I like the effect of Jquery's color animate, which transitions nicely between two colors. Yes, I know that jquery can be resource heavy, especially if you're downloading the library just for one little effect, but it's still a fun toy to play with. These aren't practical widgets as much as exploration of possibilities. Make art with the tools you have.


Colorful Circles

My first experiment with random colors. These circles (all members of the same CSS class) change to a random RGB color value. Jquery's color animate smooths the transition.

Color me random!

Mouseover the circles to see the goodness! It's the same effect as above, but on a mouseover. I like the effect of dragging the mouse back and forth through the blocks to leave a rainbow trail.

A little less random

I wanted to keep the random effect, but control the palette available to choose from. This script randomly changes the characters in this text, but the colors are picked from an array I set. The effect is applied to text in the "colorful" div class. (yeah, I know that shades of grey are not exactly colorful..but I named it before I chose those). Other text is left alone. By picking grey tones that are close together, it created a soft rippling effect. And the randomness made it seem more mysterious than if it just cycled through those colors on a loop.
