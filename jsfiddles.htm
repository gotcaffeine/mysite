Some CSS/Javascript/Jquery doodads

Lately I've been having fun with creating colorful HTML without the use of art images (code only). I was playing around with different ways of using random colors on CSS elements. Here's some of my toys.

Color me random!

Mouseover the circles to see the goodness! On mouseover change members of a div class a random color. On mouseleave, slowly change them back with animate().



Character color changer from array

Change each character in set text to a random color in an array of colors set by you. Uses jquery animate()

Colorful Circles

Create circle divs that change change to random colors using animate()


Span random color changer

Change each of your spans to a different random color.
